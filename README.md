# Personalised Webpage

## Overview
This project is an interactive Personalized Webpage built with HTML, Tailwind and JavaScript.
The webpage allows users to enter their name and age, stores the data using localStorage, and displays personalized content such as:

- A custom greeting
- A calculation from the user's birth date that gives the userâ€™s age in months
- Messages based on whether the user is over 18
- A motivational quote displayed multiple times using a loop

This project demonstrates concepts such as functions, conditionals, loops, forms, and persistent browser storage. 

---

## Technologies Used

- **HTML & CSS**  
- **Tailwind CSS** for styling  
- **JavaScript** for localStorage management
